## Probleem → Oplossing

| Probleem | NIET doen | WEL doen |
|----------|-----------|----------|
| **Muf/boomy** | Alles harder | 200Hz -2dB of HPF omhoog |
| **Scherp/harsh** | Treble omlaag | 3-4kHz -2dB |
| **Zang niet hoorbaar** | Zang omhoog | Keys/drums omlaag |
| **Geen punch** | Bass boost | 2.5kHz +1dB |
| **Te veel galm** | Negeren | Korg reverb checken |
| **Ruis/brom** | Fader omhoog | Gain omlaag, DI ground lift |

## PANIC: Feedback!

```
1. MUTE verdachte kanaal (snelste fix)
2. Of: Fader -10dB
3. Identificeer: welke mic?
4. Gain -3dB op die mic
5. Unmute, check
6. Nog steeds? → EQ cut bij feedback freq
```

**Typische feedback frequenties:**
- 250Hz = boomy/woofy
- 800Hz = honky/nasal
- 2.5kHz = harsh
- 4kHz = scherp

---

## PANIC: Clipping!

```
1. Fader -6dB (direct)
2. Gain -3dB
3. Fader terug naar 0dB
4. Check: peaks @ -12dB op meter
```

## Monitor Requests

| Zanger vraagt | Actie |
|---------------|-------|
| "Meer mezelf" | AUX 3 send +3dB op hun mic |
| "Meer drums" | AUX 3 send +3dB op Drums+Bass |
| "Minder keyboard" | AUX 3 send -3dB op Keys |
| "Alles harder" | NIET doen → vraag wat specifiek |
| "Ik hoor mezelf niet" | Check of AUX 3 send aan staat |

**Tip:** Kleine stappen (+/- 3dB), vraag "beter zo?"

---

## Soundcheck Stappenplan (5 min)

```
□ 1. Alle faders op 0dB
□ 2. Keys: speel, check meters (-12dB peak)
□ 3. Drums+Bass: check meters
□ 4. Mics: één voor één, gain tot -12dB peak
□ 5. Monitors: "hoor je jezelf?" per zanger
□ 6. Snapshot opslaan
```

---

## Nieuwe Zanger Mic Setup

```
1. Gain op +15dB (startpunt)
2. Zanger zingt normaal volume
3. Gain aanpassen tot peaks @ -12dB
4. HPF @ 100Hz AAN
5. Compressor: Threshold -18dB, Ratio 3:1
6. AUX 3 send op -6dB (monitor)
7. Vraag: "hoor je jezelf in de monitor?"
```
